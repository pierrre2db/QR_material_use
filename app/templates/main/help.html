Aide et support

Comment utiliser le système

Créer une session
  1. Connectez-vous avec votre compte enseignant
  2. Cliquez sur "Créer une session" dans votre tableau de bord
  3. Sélectionnez l'équipement que vous souhaitez utiliser
  4. Une session sera créée avec un QR code dynamique
  5. Partagez ce QR code avec vos étudiants pour qu'ils puissent s'enregistrer
Scanner un équipement
  1. Utilisez l'application mobile ou l'appareil photo de votre téléphone
  2. Scannez le QR code statique de l'équipement
  3. Une session sera automatiquement créée

Scanner une session
  1. Connectez-vous avec votre compte étudiant
  2. Demandez à votre enseignant de vous montrer le QR code de la session
  3. Utilisez l'application mobile ou l'appareil photo de votre téléphone pour scanner le QR code
  4. Votre présence sera automatiquement enregistrée
Consulter votre historique
  1. Connectez-vous à votre compte
  2. Votre tableau de bord affiche automatiquement vos scans récents

Gérer les équipements
  1. Connectez-vous avec votre compte administrateur
  2. Cliquez sur "Équipements" dans la barre de navigation
  3. Vous pouvez ajouter, modifier ou supprimer des équipements
  4. Pour chaque équipement, vous pouvez générer et imprimer son QR code statique
Consulter les statistiques
  1. Votre tableau de bord affiche un résumé des statistiques
  2. Vous pouvez voir le nombre d'équipements, d'utilisateurs et de sessions
  3. Vous avez accès à toutes les sessions et tous les logs de scan

Besoin d'aide supplémentaire?

Si vous rencontrez des problèmes ou avez des questions, veuillez contacter l'administrateur système.
